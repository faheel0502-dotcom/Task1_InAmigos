# InAmigos Foundation - NGO Awareness Webpage (Task 1)

This project is a clean, modern, and fully responsive awareness webpage developed for **InAmigos Foundation** as part of the Web Development Internship program.

## 🎯 Project Objective
Create a professional and user-friendly webpage to spread awareness about the NGO's mission, ongoing social projects, active initiatives, and ways for the community to get involved or support them.

---

## ✨ Features
1. **Header & Navigation**: A modern sticky navbar with glassmorphism effects featuring the official logo and links to page sections.
2. **Hero Section**: A bold introduction banner with clear value propositions ("Empowering Lives, Restoring Hope") and calls-to-action (CTAs).
3. **About Us Section**: Detailed background highlighting InAmigos Foundation's certifications (12A, 80G, CSR-1, and ISO 9001:2015) to build trust.
4. **Key Projects & Initiatives**: Structured cards highlighting active projects with real-life drive images:
   - **Project Bachpanshala**: Child education & mentorship.
   - **Project Seva**: Hunger alleviation & clothing drives.
   - **Project Udaan**: Women's empowerment & skill training.
   - **Project Prakriti**: Environment conservation & tree planting.
   - **Project Jeev**: Stray animal welfare & rescue.
   - **Project Vikas**: Youth employability & skill development.
5. **Real-time Impact Metrics**: Highlights statistics representing key contributions made by the foundation (meals served, kids mentored, trees planted, animals rescued).
6. **Volunteer & Query Form**: Interactive form for interested volunteers and supporters to connect with the team.
7. **Footer**: Quick navigation links, certification references, and social media links.

---

## 🛠️ Technologies Used
- **HTML5**: Semantic elements for clean structure and accessibility.
- **CSS3 (Vanilla)**: For clean layouts (Flexbox, Grid), animations, transitions, and media-query responsiveness.
- **Google Fonts**: *Plus Jakarta Sans* (headings) and *Inter* (body text).
- **FontAwesome**: Modern icons for visual clarity.
